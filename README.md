# wsmusic 
# 需要本地先部署 网易云音乐 Node.js API service
# Github:https://github.com/Binaryify/NeteaseCloudMusicApi

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
